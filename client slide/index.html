Instruksi Game

  1. Masukkan username dan pilihlah level
  2. Level memengaruhi gerakan musuh menjadi bertambah speed
  3. Arrow left, arrow right, arrow top, arrow down untuk menggerakan pesawatnya
  4. Tekan Space untuk menembak musuh
  5. Kamu hanya memiliki nyawa sebanyak 3 nyawa
  6. Tekan Escape untuk pause game
  7. Enjoy the game!
